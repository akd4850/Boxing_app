# Boxing App UI 설계

## 타이머 화면

```
           1 / 12 ROUND

            ┌────────┐
            │        │
            │  0:00  │
            │        │
            │ [▶시작] │   ← 시작 누르면 [⏸일시정지]로 변경
            │        │
            └────────┘

            [ 리셋 ]

       [ 운동시간 설정 ]
       [ 프리셋 불러오기 ]
```

### 구성 요소

| 요소 | 설명 |
|---|---|
| 라운드 표시 | 현재 라운드 / 총 라운드 (예: 1 / 12 ROUND) |
| 원형 타이머 | 큰 원 안에 시간 표시 (0:00) |
| 시작/일시정지 버튼 | 타이머 원 안, 시간 아래 위치. 시작 ↔ 일시정지 토글 |
| 리셋 버튼 | 타이머 원 아래 위치 |
| 운동시간 설정 | 라운드 시간, 휴식 시간, 총 라운드 수 설정 |
| 프리셋 불러오기 | 저장된 타이머 설정 불러오기 |
