# Boxing App 기획서

## 1. 앱 소개

- **앱 이름**: Boxing App (가칭)
- **목적**: 복싱 트레이닝 시 운동 기록과 타이머를 간편하게 사용
- **타겟 사용자**: 복싱을 하는 일반 운동인
- **플랫폼**: Android, iOS

---

## 2. 핵심 기능 정의

### 2-1. 트레이닝 타이머
- 라운드 타이머 (예: 3분 운동)
- 휴식 타이머 (예: 1분 휴식)
- 라운드 수 설정
- 시작 / 일시정지 / 리셋
- 라운드 전환 시 알림 (소리, 진동)

### 2-2. 운동 기록
- 날짜별 운동 기록 저장
- 기록 항목: 날짜, 운동 종류, 라운드 수, 메모
- 기록 목록 조회
- 기록 수정 / 삭제

### 2-3. 다국어 지원
- 지원 언어: 한국어, 영어, 일본어, 중국어
- 기기 언어 설정에 따라 자동 적용
- 설정 화면에서 수동 언어 변경 가능

---

## 3. 화면 구성

| 화면 | 설명 |
|---|---|
| 홈 | 타이머, 운동 기록 메뉴 진입 |
| 타이머 | 라운드/휴식 시간 설정, 타이머 실행 |
| 운동 기록 작성 | 운동 내용 입력 및 저장 |
| 운동 기록 목록 | 날짜별 기록 리스트 |
| 운동 기록 상세 | 개별 기록 조회 / 수정 / 삭제 |
| 설정 | 기본 라운드/휴식 시간, 알림 설정, 언어 선택 |

---

## 4. 데이터 구조

### 4-1. 운동 기록 (WorkoutRecord)
| 필드 | 타입 | 설명 |
|---|---|---|
| id | int | 고유 ID |
| date | DateTime | 운동 날짜 |
| workoutType | String | 운동 종류 (예: 샌드백, 스파링, 미트) |
| rounds | int | 라운드 수 |
| memo | String | 메모 |

### 4-2. 타이머 설정 (TimerSetting)
| 필드 | 타입 | 설명 |
|---|---|---|
| roundTime | int | 라운드 시간 (초) |
| restTime | int | 휴식 시간 (초) |
| totalRounds | int | 총 라운드 수 |
| alertSound | bool | 소리 알림 여부 |
| alertVibration | bool | 진동 알림 여부 |
